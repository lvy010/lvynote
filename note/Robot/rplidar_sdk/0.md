# docs：rplidar_sdk

`rplidar_sdk` 是一个 **C++ SDK**，促进与 Slamtec RPLIDAR 传感器的*高级交互*。

允许开发人员轻松**控制激光雷达**（例如，开始扫描、获取设备信息）和**获取扫描数据**，同时抽象掉底层*硬件通信的复杂性*和*数据解释*。
